# 智能汽车项目

基于 OpenCV 的智能小车视觉识别、目标定位与转向控制项目。

## 项目简介

本项目旨在实现一个智能小车视觉识别系统，让小车能够通过摄像头识别目标物体的位置，并根据目标位置自动进行转向与移动控制。

当前阶段主要完成内容包括：

- 摄像头图像采集
- 单一颜色识别
- 轮廓提取
- 目标中心点计算
- 左 / 中 / 右位置判断
- 后续小车转向控制
- 接入AI模型，实现自动化控制

## 目录说明

### vision/
放摄像头识别代码，例如颜色识别、轮廓提取、中心点计算、左中右判断等。

### control/
放小车控制代码，例如前进、后退、左转、右转、停止等。

### docs/
放项目进度、学习笔记、函数说明、测试记录等文档。

### tests/
放测试图片、测试视频、识别截图等素材。
